# Stream Case Game v6

Что нового:
- основной экран теперь показывает этапы и кнопки переключения раундов
- экстра-этапы выглядят отдельно: золотой фон и яркий стиль
- админ-панель скрыта и открывается отдельно по кнопке
- обычные этапы и экстра-этапы настраиваются отдельно
- бомба и галочка не имеют отдельной логики в коде: это обычные предметы с собственной картинкой и шансом
- открытые кейсы сохраняются внутри текущего этапа, пока ты не начнёшь новую игру или не закроешь кейсы этапа вручную

Что заменить в репозитории:
- index.html
- style.css
- app.js
- config.json

Как пользоваться:
1. Открой сайт.
2. На основном экране переключай этапы кнопками `Назад` / `Следующий этап`.
3. Кнопка `Завершить этап` только отмечает этап как завершённый.
4. В `Админ-панели` можно отдельно редактировать обычные этапы и экстра-этапы.
5. Кнопка `Кейс → во все кейсы этапа` копирует выбранный кейс на весь этап.
6. Кнопка `Этап → во все этапы этого типа` копирует текущий этап во все обычные или во все экстра-этапы.

Важно:
- импорт `config.json` поддерживается
- экспорт `config.json` поддерживается
- картинки можно добавлять ссылкой или загрузкой JPG/JPEG/PNG/WebP
