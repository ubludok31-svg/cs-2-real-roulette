# Case Roulette

Обновлённая версия:
- 16 кейсов в сетке 4×4
- кейсы меньше по размеру, чтобы больше помещалось на экран
- после обновления страницы все кейсы снова закрываются
- админ-панель скрыта и открывается кнопкой
- у предметов можно задавать название, редкость, вес и фон-картинку
- JPG/JPEG можно загружать прямо через админку

## Как обновить на GitHub

Замени в репозитории файлы:
- `index.html`
- `style.css`
- `app.js`
- `README.md`

`config.json` можно не менять, но можно тоже заменить на новый, если хочешь чистый старт.

## Как пользоваться

1. Открой сайт.
2. Нажми `Админ-панель`.
3. Выбери кейс.
4. Добавь предметы и картинки.
5. Закрой админку.
6. Открывай кейсы.
7. Если обновишь страницу, все кейсы станут снова закрытыми.
